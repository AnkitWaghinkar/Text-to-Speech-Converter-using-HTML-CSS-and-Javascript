# Text-to-Speech Converter

## Description
A text-to-speech converter built with HTML, CSS, and JavaScript. It converts written text into speech.

## Features
- Convert any text to speech.
- Feature to change the voices.
- Simple interface.

## Technologies Used
HTML, CSS and Javascript
